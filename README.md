# Addons by Calisto 3D

Website hub for Blender add-ons by **Calisto 3D**, a 3D artist creating tools to speed up Blender and asset workflows.

This repository is used to host landing pages for current and future Blender add-ons through GitHub Pages.

- **X:** [@calisto3d](https://x.com/calisto3d)

## Pages

- **UEFN Exporter:** `uefn-exporter/index.html`
- **Batch Exporter:** `batch-exporter/index.html`

```text
Websites/
├── index.html
├── uefn-exporter/
│   └── index.html
├── batch-exporter/
│   └── index.html
└── README.md
```

## GitHub Pages setup

1. Go to **Settings > Pages**.
2. Under **Build and deployment**, choose **Deploy from a branch**.
3. Select the `main` branch.
4. Select the `/root` folder.
5. Save and wait for GitHub Pages to publish the site.

The website should be available at:

```text
https://calisto3d.github.io/Websites/
```

Addon pages should be available at:

```text
https://calisto3d.github.io/Websites/uefn-exporter/
https://calisto3d.github.io/Websites/batch-exporter/
```

## Adding a new addon page

1. Create a new folder using a clean lowercase name, for example:

```text
new-addon/
```

2. Add an `index.html` file inside that folder.
3. Add a new button/link on the main `index.html` page.
4. Commit the changes to the `main` branch.
