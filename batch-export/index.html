Features

Everything you need.

📦

Separate files

Each selected object is exported as its own file, named after the object automatically.

🔗

Merged export

Toggle "Export as one file" to combine all selected objects into a single export.

🗂️

Format selector

Switch between FBX, OBJ, GLTF and STL with a single dropdown — no settings buried in menus.

📁

Folder picker

Browse to any folder on your system directly inside Blender. No hardcoded paths.

Installation

Up in 30 seconds.

1

Download the file

Grab the latest .py file from the Releases page on GitHub.

2

Install in Blender

Go to Edit → Preferences → Add-ons → Install and select the downloaded file.

3

Enable it

Tick the checkbox next to Batch Exporter in the add-ons list.

4

Open the panel

Press N in the 3D Viewport and find the Batch Export tab on the right.
