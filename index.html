UEFN Workflow

Built for organized asset output

UEFN Exporter keeps the bake and export steps separate: textures are generated during baking, while the export step sends only the mesh to the asset folder.

1. PrepareRename selected meshes and fix pivots to bottom-center.
2. BakeGenerate the baked texture maps using Cycles.
3. Create baked meshDuplicate the mesh, apply baked material and keep only the bake/export UV.
4. ExportBatch export selected meshes into individual folders.

Installation

Install any add-on

The process is the same for most Blender add-ons packaged as a .zip or .py file.

1

Download

Open the add-on release page and download the latest file.

2

Install

In Blender, go to Edit → Preferences → Add-ons → Install.

3

Enable

Search for the add-on name and activate the checkbox.

4

Open panel

Press N in the 3D Viewport and open the add-on tab.
