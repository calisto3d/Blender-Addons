Features

Everything you need for UEFN

Stop doing things manually. UEFN Exporter handles the full pipeline — from raw Blender models to baked textures and import-ready mesh folders for multiple selected assets at once.

✏️

Smart Rename

Automatically names your meshes following the Unreal Engine convention. Supports custom prefix and folder name. Special characters and lowercase starts are sanitized automatically.

SM_AssetName1, 2, 3…
🌍

Pivot Fix

Sets the origin to the bottom-center of each mesh's bounding box and moves the object to world origin (0, 0, 0). No more off-center pivots when importing into UEFN.

Critical for UEFN import
🎨

PBR Texture Bake

Bake Albedo, Normal, Roughness, Metallic, AO, Emissive and Opacity maps in one click. Choose resolution, control samples, format and bake padding to reduce UV bleeding.

Cycles powered · Padding control
🗺️

UV Modes

Bake into your existing UV map or let the addon generate a Smart UV Project automatically — with configurable angle limit, island margin and Scale to Bounds.

Existing UV · Smart Project · Scale to Bounds
🪄

Auto Baked Mesh

After baking, the addon can automatically duplicate the object, create a clean Principled BSDF material wired to the baked textures, hide the original and keep only the bake/export UV on the new mesh.

Baked mesh workflow
📦

Batch Per-Asset Export

Export multiple selected meshes at once. Each mesh is exported into its own SM_AssetName folder. Export is mesh-only, so baked textures are not copied again into the export folder.

Batch export · Mesh only
💾

Presets

Save your full workflow configuration as a named preset and replay it with Run All Steps. Enable or disable individual steps (rename, pivot, bake, export) per preset.

One-click automation

Workflow

From Blender to UEFN in 5 steps

Run each step independently or automate everything with a preset.

1

Rename

Select objects, set folder name. The addon renames to SM_Folder1, SM_Folder2…

2

Fix Pivot

Origin is placed at the bottom-center of the mesh and the object moves to world (0,0,0).

3

Bake

PBR maps, including Opacity, are baked via Cycles. Control resolution, samples, format and bake padding.

4

Baked Mesh optional

Duplicate with baked material wired to BSDF, hide the original and keep only the bake UV on the new mesh.

5

Export

Export multiple selected meshes at once. Each mesh goes into its own SM_AssetName folder, while baked textures stay in the bake output location.

Output

Clean folder structure

Batch export sends each selected asset to its own folder containing only the mesh. Baked textures are generated during the bake step and are not copied again during export.

Installation

Install in 3 steps

Requires Blender 3.6 or newer. No Python dependencies needed.

1

Download the addon

Click Download Addon above and save uefn_exporter.zip anywhere on your computer.

2

Install in Blender

Open Blender → Edit > Preferences > Add-ons → click Install → select the .zip → activate the checkbox next to UEFN Exporter.

3

Open the panel

In the 3D Viewport, press N to open the Sidebar and click the UEFN Exporter tab. Done! 🎉

💡 Bake requires Cycles — make sure Render Properties → Render Engine is set to Cycles before baking. The addon can generate a Smart UV Project automatically with Scale to Bounds and adjustable Island Margin.
